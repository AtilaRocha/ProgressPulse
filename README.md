# ProgressPulse
ProgressPulse é um dashboard intuitivo para gerenciar e acompanhar o progresso de metas. Com funcionalidades de autenticação, CRUD de metas e gráficos dinâmicos, oferece uma experiência responsiva e escalável. Desenvolvido com React, NestJS, Docker e PostgreSQL, é ideal para quem busca foco e resultados efetivos.
